# How to apply css

```
// Use important for responsive designs

@media only screen and (min-width: 768px) {
  h1 {
    font-size: 5rem !important;
    color: blue;
  }
}

@media only screen and (max-width: 768px) {
  h1 {
    font-size: 1rem !important;
    color: red;
  }
}
```
